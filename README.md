# Plan Management System

## Overview
This project is a comprehensive plan management system designed to facilitate strategic planning, goal setting, and roadmap execution for teams and organizations.

## Features
- **Strategic Planning**: Define long-term visions and mission statements.
- **Goal Setting**: Create SMART goals and objectives (OKRs).
- **Roadmap Visualization**: Visual timelines and Gantt charts for project phases.
- **Resource Allocation**: Manage team capacity and resource distribution across plans.
- **Milestone Tracking**: Monitor key deliverables and critical path achievements.

## Getting Started

### Prerequisites
- Python 3.8+
- PostgreSQL

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application server:
```bash
python manage.py runserver
```
