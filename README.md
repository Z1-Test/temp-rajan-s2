# Task Management System

A simple and efficient task management application to help you organize your work and life.

## Features

- **Create Tasks**: Easily add new tasks with titles and descriptions.
- **Track Progress**: Mark tasks as pending, in-progress, or completed.
- **Prioritize**: Assign priority levels to your tasks.
- **Organize**: Group tasks by categories or projects.
