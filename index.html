Introduction

Staylook is a design system focused on being Curved, Editorial, Minimal, and Selective Expressive. Built for developers who value clean implementation and aesthetic excellence.

Design Principles

  • Curved: Soft, rounded edges throughout. No sharp corners.
  • Editorial: Clean layouts with strong typographic hierarchy.
  • Minimal: Every element serves a purpose.
  • Expressive: Selective color usage to guide attention.

The Expressive Usage Rule

Expressive styling is NOT required on every page. Most of your UI should use Standard tokens. Only use Expressive for explicitly marked "Special Moments".

Design tokens are the foundation of Staylook. They ensure consistency across all components.
